## 🚀 Project Overview

AITAT NetMap is a system for visualizing and managing network infrastructure in an interactive way.

It allows users to monitor devices, view network topology, track status, and manage incidents.

The project consists of:
- Flutter frontend (mobile / web)
- Node.js backend API
- MongoDB database

---

## 🎯 Features

- Interactive network map
- Device visualization (PC, switches, printers, etc.)
- Device status monitoring
- Ping / connectivity check simulation
- Incident management system
- Event logging system
- Room-based device grouping
- Refresh system for updates

---
